About Ships - ROOT

A root POM having: - constants defined (which are version numbers mostly) - dependency and plug-in management configured - reporting configured

Project Modules

This project has declared the following modules:

Name Description
Ships - CLIENT Ships game client. An isolated module having a light-weight client logic and GUI (JavaFX). The module packaging is defined independently.
Ships - SERVER Ships game server. An isolated module being mostly responsible for, - handling a network connection - handling message exchange - handling heavy-weight game rules The module packaging is defined independently.
Ships - SHARED Used both by client and server modules. An isolated module having: - fleet model objects - communication API + core
